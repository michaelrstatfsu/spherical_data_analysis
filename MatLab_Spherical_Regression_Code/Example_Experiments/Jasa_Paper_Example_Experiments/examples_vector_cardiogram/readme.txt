This contains data modules using vector cardiogram data.

To run this you will need the vcg.xls file from 
http://www.mcs.st-and.ac.uk/~pej/vcg.html

Run Fetch_Vector_Cardiogram_Data.m in this directory to automatically download, construct, and save the data sets. This will automatically download the vcg.xls file if you are missing it in the matlab search directory.
