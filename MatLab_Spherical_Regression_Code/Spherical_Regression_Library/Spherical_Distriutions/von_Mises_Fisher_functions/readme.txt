This directory contains code for hyperspherical analysis using the von Mises Fisher distribution written by Michael Rosenthal and Sungkyu Jung.

Michael_Rosenthal_vmf_code
This directory contains functions for computing likelihoods and for estimating the concentration parameter kappa.
http://www.stat.fsu.edu/~michaelr

Sungkyu_Jung_vmf_code
This directory contains functions for simulating from the von-Mises Fisher distribution and a generalized Rodrigues' rotation formula for computing rotation matrixes betweens unit vectors.
http://www.unc.edu/~sungkyu/MiscPage.html
