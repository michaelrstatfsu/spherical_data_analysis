Magsat data and its documentation can be found at 
ftp://spdf.gsfc.nasa.gov/pub/data/magsat/
It is suggest that you save the files from this site before running
Collect_magsat_data which will randomly take 15 observations from the each of the first 10 days of available data and combine the data into a data set. 

X denotes the geographical location
Y denotes the direction of magnetic force
T denotes the time index (in matlab time index format)

magsat_79_11_12_to_79_11_21.mat has these values stored for future applications. 

See Example01.m for a plot of the data.
